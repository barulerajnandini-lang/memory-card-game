# 🧠 Memory Card Game

A simple and interactive **Memory Card Matching Game** built using **HTML, CSS, and JavaScript**.

Players enter their name on the login page and try to match all the card pairs in the fewest moves possible.

---

## 🚀 Live Demo

🌐 https://barulerajnandini-lang.github.io/memory-card-game/

---

## 📸 Screenshots

### Login Page
- Player enters their name.
- Clicks **Start Game** to begin.

### Game Page
- Match all the card pairs.
- Track your moves.
- Restart the game anytime.
- Logout to return to the login page.

## ▶️ How to Run

1. Clone the repository

```bash
git clone https://github.com/barulerajnandini-lang/memory-card-game.git
```

2. Open the project folder.

3. Open **index.html** in your browser.

4. Enter your name.

5. Start playing!

---

## 🎯 Game Rules

- Enter your name on the login page.
- Click cards to reveal emojis.
- Match two identical cards.
- Match all 8 pairs to win.
- Try to finish in the fewest moves.
